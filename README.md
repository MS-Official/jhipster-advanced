# JHipster Platform Starter Pack

This package is an all-in-one specification and template pack for a JHipster-based enterprise starter.

## What is included
- JDL templates for monolith, gateway, and microservices
- OTP schema and service boundaries
- Docker Compose and Kubernetes manifest matrix
- Jenkins pipeline design
- n8n workflow catalog
- JMeter test suite map
- Maven and Gradle starter module maps
- npm packaging guidance for publishing this pack

## Intended use
1. Copy this repository.
2. Rename the npm scope/package.
3. Implement the Yeoman/JHipster blueprint under `blueprint/`.
4. Publish the package to npm.
5. Generate apps with your conventions and import the JDL bundles.
